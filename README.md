# YouRics (Youtube & Lyrics)

**Web to listen to songs and read lyrics.**

### `npm run dev`
Runs the app in the development mode.<br />
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## Modules used
 - React-router-dom  
 - Nodemon
 - axios
 - Googleapis
 - mysql2
 - sequelize
 - sequelize-cli (global)
 - puppeteer
 - bootstrap
 - react-bootstrap
 - @material-ui/core
